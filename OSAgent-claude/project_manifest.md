# MCP Self-Healing Bridge

## Project Vision
To transform traditional "Passive Monitoring" into "Active AI-Ops." This project implements a Model Context Protocol (MCP) server that empowers LLMs to act as autonomous SREs within a restricted, safe execution environment.

## Core Features
- **Context Awareness:** Injects SLA/SLO policies directly into the LLM's context window.
- **Safety First:** Uses a tool-calling schema to prevent command injection.
- **Extensible:** Easily hooks into Proxmox, Kubernetes, or Cloud APIs.
